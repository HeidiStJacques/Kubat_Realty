# 🏡 Kubat Realty Website

A modern, responsive real estate website built for Kubat Realty to showcase listings, services, and brand presence.

Designed with a focus on clean layout, strong visual hierarchy, and a professional user experience for buyers and sellers.

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Hero image carousel for branding and featured properties
- Clean navigation structure:
  - Home
  - About
  - Areas Served
  - Listings
  - Buy
  - Sell
  - Contact
- Call-to-action sections to drive inquiries
- Professional layout with consistent branding
- Scalable structure for future MLS/IDX integration

---

## 🎨 Design Approach

- **Primary color:** Deep navy (trust & professionalism)
- **Accent color:** Gold (calls-to-action and hover states)
- **Layout:** Clean, spacious, and easy to navigate
- **Goal:** Build credibility and encourage client engagement

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **UI Components:** Swiper (carousel)
- **Hosting:** AWS (planned)
